Summary: Latitude vs. X

The purpose of this project was to analyze changes in weather in relation to location based on latitiude for a random sample of cities. We hypothesized that there was a possible correlation between a given city's latitude relative to the equator and its resulting weather factors. To accomplish this analysis, we used OpenWeatherMap API as our source of data to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib's graphing capabilities to plot the Latitude of Cities versus various weather factors. Factors we looked at included: temperature, cloudiness, wind speed, and humidity. This site provides the raw weather data per city and visualizations created as part of the analysis, as well as explanations and descriptions of any trends observed in the analysis.
